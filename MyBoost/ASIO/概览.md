Containers：Asio  
Portable networking and other low-level I/O, 
including sockets, timers, hostname resolution, 
socket iostreams, serial ports, file descriptors 
and Windows HANDLEs.

Author(s) ：Chris Kohlhoff
First Release： 1.35.0
Standard： 
Categories：Concurrent Programming, Input/Output

使用g ++ 3.3或更高版本的Linux（2.4或2.6内核）。
